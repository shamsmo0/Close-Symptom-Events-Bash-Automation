# Incident Event Closer - Bash Automation Tool

A lightweight Bash script that automates the closure of symptom events by linking incident tickets to their corresponding events. Built for system monitoring and event management workflows.

##  Features

-  Incident validation from ticket status file
-  Automatic event ID mapping from incident numbers
-  Event status checking before closure
-  Append closed events to a separate output file
-  Error handling for missing incidents
-  Clear console output for each scenario

## Tech Stack

| Tool | Purpose |
|------|---------|
| Bash | Core scripting language |
| awk | Text parsing and field extraction |
| while loops | Iterative file processing |
| read | Interactive user input |
